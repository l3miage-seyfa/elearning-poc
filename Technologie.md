## 1. Docebo — LMS Enterprise augmenté par IA

**Site** : docebo.com | **Cible** : Grandes entreprises (B2B)

### Ce que fait Docebo avec l'IA

Docebo a développé un écosystème IA appelé **Harmony** qui couvre plusieurs couches :

| Module IA | Ce que ça fait |
|-----------|---------------|
| **Harmony (assistant IA)** | Répond aux questions des apprenants en langage naturel, fait de la recherche sémantique dans le catalogue de cours, guide les admins pas à pas |
| **Harmony Tutor** | Tuteur personnel intégré dans chaque cours : explique les concepts, résume le contenu, aide l'apprenant pendant la formation |
| **Content Creator (IA)** | Génère des cours complets (vidéos, simulations, quiz) à partir d'un PDF, PPT, texte ou prompt. **80 à 99% de temps économisé** en création |
| **AI Roleplay** | Simulations interactives avec avatars IA pour pratiquer des situations réelles (entretiens, vente, etc.) |
| **Skills Intelligence** | Analyse les compétences des employés, identifie les gaps et recommande des formations ciblées |
| **Content Discovery** | L'apprenant pose une question en langage naturel → l'IA recommande les cours pertinents |
| **AgentHub** | Agents IA autonomes (en cours de déploiement 2026) |

### Cas d'usage typiques (Docebo)

1. **Onboarding collaborateur** : Un nouveau salarié arrive → l'IA génère automatiquement un parcours personnalisé selon son poste
2. **Formation compliance** : Un employé demande à Copilot/ChatGPT une règle de conformité → Harmony répond sans quitter l'outil
3. **Création de contenu interne** : Un expert métier uploade un PDF de procédure → Docebo Creator génère un cours interactif avec quiz en quelques minutes
4. **Sales enablement** : Les commerciaux s'entraînent via des simulations IA (roleplay) avant des appels clients
5. **Upskilling** : L'IA identifie les compétences manquantes d'un employé et lui recommande les modules à suivre

### Chiffres clés (cas clients)
- 800h économisées/an (Booking.com)
- $1,5M économisés en coûts de formation (KCF Technologies)
- -80% overhead admin (création de contenu)
- 135 000€ économisés sur l'onboarding/an (Flix)

## 2. CoorpAcademy (by Go1) — LXP axée engagement

**Site** : coorpacademy.com | **Cible** : Entreprises mid-market, formations soft skills

### Ce que fait CoorpAcademy avec l'IA

| Fonctionnalité | Description |
|----------------|-------------|
| **Catalogue 1500+ cours premium** | Inclus dans la plateforme, couvre 80% des soft skills WEF |
| **Pédagogie inversée + Battle** | Format quiz/bataille entre collègues, très gamifié |
| **Cockpit (CMS)** | Outil de création de cours maison (SCORM, vidéo, PDF) |
| **Adaptive learning** | Adaptation du contenu selon le profil comportemental de l'apprenant (5 attitudes identifiées par leur R&D à l'EPFL) |
| **Dashboard analytique** | Suivi engagement, progrès, certifications |

### Cas d'usage typiques (CoorpAcademy)

1. **Formation soft skills à grande échelle** : Leadership, communication, gestion du temps pour des milliers d'employés
2. **Apprentissage social/collaboratif** : Les Battle permettent aux collègues de se challenger mutuellement
3. **Création de cours métiers internes** : Les RH créent des modules sur-mesure via Cockpit
4. **Montée en compétences continue** : Recommandations personnalisées selon les 5 profils comportementaux

## 3. Domoscio (acquis par Rise Up) — Adaptive Learning & Ancrage mémoriel

**Site** : domoscio.com | **Cible** : Formation professionnelle, entreprises

### Ce que fait Domoscio avec l'IA

Domoscio est spécialisé dans **l'adaptive learning scientifique** basé sur les sciences cognitives :

| Produit | Ce que ça fait |
|---------|---------------|
| **Domoscio Hub** | Diagnostique le niveau de compétences de chaque apprenant → recommande les actions de formation adaptées à son profil et ses besoins |
| **Domoscio Lock** | Détermine le profil cognitif → envoie des **rappels espacés personnalisés** pour ancrer durablement les connaissances (méthode de la répétition espacée / spaced repetition) |
| **Domoscio Lab** | Accompagnement pour développer des projets IA sur-mesure en éducation/formation |

### Cas d'usage typiques (Domoscio)

1. **Plan de formation individualisé** : Diagnostic initial → parcours adapté à chaque apprenant selon son niveau réel
2. **Ancrage post-formation** : Après une formation, l'IA envoie des micro-quiz de rappel au bon moment pour éviter la courbe de l'oubli
3. **Mesure d'impact formation** : Suivi scientifique de la rétention des connaissances dans le temps
4. **Intégration dans un LMS existant** : Domoscio se greffe sur une plateforme déjà en place

### Chiffres clés
- **37%** de temps gagné pour acquérir les compétences cibles
- **46%** de rétention supplémentaire 3 mois après formation

## 5. Synthèse : Quelles fonctionnalités sont les plus pertinentes pour un POC ?

En analysant ces 3 outils, les fonctionnalités IA les plus **utiles, démontrables et faisables** pour un POC sont :

### ✅ Fonctionnalités "coeur" du marché (ce que tous font)
- Génération automatique de quiz à partir d'un contenu
- Parcours d'apprentissage personnalisé
- Suivi de progression

### 🌟 Fonctionnalités différenciantes (valeur ajoutée forte)
- **Génération de cours depuis un document** (Docebo Creator) → très fort pour la gestion des connaissances internes
- **Chatbot pédagogique** (Docebo Harmony Tutor) → répond aux questions sur un contenu
- **Ancrage mémoriel par rappels espacés** (Domoscio Lock) → scientifiquement prouvé (+46% rétention)

### 🎯 Ce qu'un POC devrait démontrer
Un POC crédible doit reproduire **au moins une** de ces fonctionnalités différenciantes sur des données réalistes d'entreprise.
